2048 3D

Фізична 3D-реалізація механіки 2048 для Android. Архітектура побудована на Strategy Pattern — три типи кубів з різною логікою злиття (звичайний, універсальний, бомба з OverlapSphere). Spawn-система реалізована через Object Pool з автоматичним розширенням пулу.

Tower Defense

Tower Defense з процедурно згенерованою картою. Хвильова система спавну побудована на Coroutines та Object Pool. Рівні налаштовуються через ScriptableObject. Ключова особливість — кастомний інструмент в Unity Editor для автоматичної генерації шляху та розміщення тайлів.

2D Platformer

2D платформер з механікою стрільби та патрулюючими ворогами. Гравець може бігати, стрибати та стріляти в обох напрямках. Архітектура розділена на незалежні компоненти — Input, Mover, Shooter — відповідно до принципу єдиної відповідальності.
